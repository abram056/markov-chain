# Markov Chain Text Generator

A C++ implementation of a Markov Chain-based text generator that produces sequences based on probabilistic word transitions.

## How It Works
- Reads input text from a file
- Builds a state transition map of word sequences
- Generates new text based on probability distributions

## Features
- Custom starting word
- Configurable output length
- File-based input processing

## Tech Stack
- C++

## Build & Run

```bash
make
./markov
```
